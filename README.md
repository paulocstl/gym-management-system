# FITFLOW - Gym Management System

## About the Project

FitFlow is a gym management system developed in Python using SQLite database.

The system was designed to help manage the internal operations of a gym, allowing the registration and management of students, employees, membership plans, workouts, and exercises.

This project was developed as an academic assignment for a Programming course.

---

## Features

### Students

* Register students
* List students
* Search students by name
* Update student information
* Delete students

### Employees

* Register employees
* List employees
* Edit employee information
* Delete employees

### Membership Plans

* Register plans
* List plans
* Update plans
* Delete plans

### Workouts

* Register workouts
* Assign students and employees
* View workouts
* Edit workouts
* Delete workouts

### Exercises

* Register exercises
* List exercises
* Add exercises to workouts
* Remove exercises from workouts

---

## Technologies

* Python 3
* SQLite
* sqlite3 library

---

## Project Structure

* main.py → Main menu and system integration
* alunos.py → Student management
* funcionarios.py → Employee management
* planos.py → Membership plan management
* treinos.py → Workout and exercise management
* database.py → Database configuration

---

## How to Run

1. Install Python 3.
2. Clone or download the repository.
3. Open the terminal inside the project folder.
4. Run:

```bash
python main.py
```

The database will be automatically created on the first execution.
